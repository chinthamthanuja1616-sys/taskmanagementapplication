# Task Management Application

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS

## Features
- User Registration
- User Login & Logout
- Add Task
- Edit Task
- Delete Task
- Mark Task as Completed

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000
